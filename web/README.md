# 🌐 Web界面使用说明

## 快速启动

```bash
cd stock-analyzer

# 方式1: 直接启动
python3 -m streamlit run web/app.py

# 方式2: 使用启动脚本
cd web && ./start.sh
```

## 访问应用

启动后，在浏览器中打开：
```
http://localhost:8501
```

## 使用步骤

### 1. 输入股票代码
在左侧侧边栏输入股票代码：
- A股：6位数字（如：000001、600519）
- 美股：字母代码（如：AAPL、TSLA）
- ETF：ETF代码（如：510300）

### 2. 选择市场类型
- **A**: A股市场
- **US**: 美股市场
- **ETF**: ETF基金

### 3. 更新数据
点击"🔄 更新数据"按钮获取最新行情数据

### 4. 查看分析结果
- **实时行情**: 显示当前价格、涨跌幅、最高最低价
- **交易信号**: 综合信号（买入/卖出/观望）及置信度
- **技术指标**: MACD、RSI等技术指标图表
- **策略回测**: 启用回测后显示回测结果

### 5. 配置策略
在侧边栏选择要使用的策略：
- 双均线交叉策略
- MACD策略
- RSI超买卖策略

### 6. 回测
勾选"启用回测"，选择回测策略，查看历史表现

## 界面功能

### 主页面区域
1. **数据概览卡片**: 当前价格、涨跌幅、成交量等
2. **交易信号**: 综合信号和策略详情
3. **K线图**: 价格走势与均线
4. **技术指标**: MACD、RSI图表
5. **原始数据**: 历史数据表格

### 侧边栏功能
- 股票代码输入
- 市场选择
- 数据更新按钮
- 策略配置
- 回测设置

## 注意事项

1. **首次使用**: 需要下载数据，可能需要几秒钟
2. **数据更新**: 建议定期点击"更新数据"获取最新行情
3. **中文显示**: 图表已配置中文字体，显示正常
4. **免责声明**: 分析结果仅供参考，不构成投资建议

## 故障排除

### 如果中文显示不正常
确保系统已安装中文字体（如黑体、微软雅黑）

### 如果无法获取数据
1. 检查网络连接
2. 确认股票代码正确
3. 查看日志输出

### 如果启动失败
```bash
# 检查依赖
pip install -r requirements.txt

# 重新安装streamlit
pip install --upgrade streamlit
```

## 快捷键

- **R**: 重新运行分析
- **C**: 清除缓存
